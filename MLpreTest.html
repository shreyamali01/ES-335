Question 1

Answer : We have two matrices of the dimensions (n,k) and (k,m). The resultant matrix will have dimensions (n,m).

Now, k is the number of columns in first matrix and number of rows in the second matrix. Thus, every entry will require k multiplications each. Now, we have entries across n rows and m columns. Thus, the total number of multiplications involved in multiplying the two matrices would be nkm.

Similarily, for addition we need (k-1) additions for a single entry. Now, there are (nxm) elements in this matrix of size (n,m). Thus, we need nxmx(k-1) additions for matrix multiplication.

Answer : Finding the highest element in the array has a time complexity of O(n) and finding the second highest element will have a time complexity of O(2n) i.e. O(n). Similarily, using the same approach finding the median of the array will take a time of O(n^2) i.e. O(n^2).

Now, a better approach would be to sort the array and find the n/2th element in the sorted array when n is even or average of (n/2)th element and ((n/2)-1)th element when n is odd.

Thus, let us implement this with the help of a code

#median using a function
def give_median(arr):
  sorted_array = np.sort(arr)
  n = len(arr)
  if (n%2) == 0:
    a = sorted_array[n//2]
    b = sorted_array[(n//2)-1]
    c = (a+b)/2
    return c
  else:
    return sorted_array[n//2]

start2 = time.time()
y = give_median(arr)
stop2 = time.time()
print('The median using the above given function is',y, 'and the time taken is',(stop2-start2))

Here, we observe that the time taken by the function numpy.median is less than that of the time taken by the custom function. This is because numpy operations are vectorized and they not access each element individually.

Answer : To find the gradient, let us find the partial derivative of the given function.

partial derivative of the function wrt x is

2xy + y^3*cos(x)

partial derivatice of function wrt y is

x^2 + 3y^2sin(x)

(2xy + y^3cos(x), x^2 + 3y^2*sin(x))

#computing gradient using jax

import jax
import jax.numpy as jnp
from jax import grad
import math

#defining x and y
x0 = np.random.rand()
y0 = np.random.rand()

#converting to jax array for simplicity
x = jnp.array(x0)
y = jnp.array(y0)

#getting gradients using jax
dfx = grad(lambda x: (x**2 * y) + (y**3 * jnp.sin(x)))(x)
dfy = grad(lambda y: (x**2 * y) + (y**3 * jnp.sin(x)))(y)

#getting gradient using analytical solution
dfx_a = 2*x0*y0 + y0**3*math.cos(x0)
dfy_a = x0**2 + 3*y0**2*math.sin(x0)

print("jax sol : The gradient of the given function is (", dfx, "," ,dfy,")")
print("analytical sol : The gradient of the given function is (", dfx_a, "," ,dfy_a,")")

analytical sol : The gradient of the given function is ( 0.23784282356490347 , 0.8981527171163322 )

def find_argmin(arr):
  ind = 0
  min = arr[0]
  for i in range(len(arr)):
    if arr[i] < min:
      min = arr[i]
      ind = i

  return ind


x = find_argmin(arr)
print('by creating a function',x)

y = np.argmin(arr)
